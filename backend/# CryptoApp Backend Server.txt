# CryptoApp Backend Server

Este es el servidor backend para CryptoApp, una plataforma de seguimiento de inversiones en criptomonedas.

## Descripción

CryptoApp permite a los usuarios realizar un seguimiento de sus inversiones en criptomonedas, visualizar transacciones, y ver el rendimiento de sus carteras.

## Funcionalidades Principales

- Crear, actualizar y eliminar usuarios
- Realizar transacciones de criptomonedas
- Obtener información detallada sobre el perfil de un usuario, incluyendo balance, transacciones y valor total de la cartera en USD
- Obtener una lista de todos los usuarios con información detallada

## Tecnologías Utilizadas

- Node.js
- Express.js
- MongoDB
- Mongoose (ODM para MongoDB)

## Instalación

1. Clona el repositorio: `git clone https://github.com/tu-usuario/CryptoApp-Backend.git`
2. Instala las dependencias: `npm install`
3. Configura las variables de entorno en un archivo `.env` (ver `example.env` para un ejemplo)
4. Inicia el servidor: `npm start`

## Endpoints API

- `GET /api/users`: Obtiene una lista de todos los usuarios con información detallada.
- `GET /api/users/:userId`: Obtiene información detallada sobre un usuario específico.
- `POST /api/users`: Crea un nuevo usuario.
- `PUT /api/users/:userId`: Actualiza la información de un usuario existente.
- `DELETE /api/users/:userId`: Elimina un usuario.
- `POST /api/users/:userId/transactions`: Realiza una nueva transacción para un usuario específico.
